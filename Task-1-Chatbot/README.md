# Smart Rule-Based Chatbot

## Objective
Develop a rule-based chatbot using Python that can interact with users and respond based on predefined rules.

## Features
- Greets users
- Responds to common questions
- Tells jokes and facts
- Shows current date and time
- Performs simple calculations
- Provides personalized responses
- Exits gracefully when requested

## Technologies Used
- Python 3

## How to Run

1. Install Python on your system.
2. Download or clone the repository.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python chatbot.py
```

## Sample Commands

- Hi
- How are you?
- Tell me a joke
- Tell me a fact
- What is the time?
- What is today's date?
